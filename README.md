## 📖 About This Repository

Welcome to my **Node.js Playground**—a chaotic but beautiful collection of various practice projects exploring the depths of the JavaScript runtime.

From **Event Loop mechanics** to **File System manipulation**, **Stream processing**, and **CLI tool building**, this repo is where I turn coffee into code and `console.log()` into pure art.

> *"One does not simply master Node.js without accidentally deleting `node_modules` at least once."* — Every Developer Ever

## 🧠 Concepts Covered (The Real Treasure)

This repository dives deep into the core pillars of Node.js:

| Concept | What You'll Explore |
| :--- | :--- |
| 🗄️ **File System (fs)** | Reading, writing, watching, and organizing files like a digital librarian. |
| 🌊 **Streams & Buffers** | Handling massive datasets without choking your RAM. |
| 🔁 **Event Loop** | The heartbeat of Node—understanding the Poll, Timers, and Check phases. |
| 🧵 **Thread Pool** | When async isn't enough, and you need to flex Libuv's muscles. |
| 🛠️ **CLI Tooling** | Parsing `process.argv`, building global commands, and shipping utilities. |
| 🌐 **HTTP Servers** | Bare-metal `http` modules and routing without frameworks. |
| 🗺️ **Path Manipulation** | Cross-platform path resolution that actually works. |

1. **Clone the repository** (like a Sharingan copy):
```bash

```

2. **Install dependencies** (Chakra Mode):
